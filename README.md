# 🤖 Basic Chatbot in Python

A simple command-line chatbot built using Python that interacts with users through predefined responses. This project demonstrates the fundamentals of Python programming, user input handling, conditional statements, and conversational logic.

## 📌 Project Overview

The Basic Chatbot allows users to communicate with a chatbot through text-based conversations. It responds to common greetings and queries, making it an excellent beginner-friendly project for learning Python.

## ✨ Features

* 💬 Interactive text-based conversation
* 👋 Responds to greetings and common questions
* ⚡ Fast and lightweight
* 🧠 Simple rule-based response system
* 🚪 Exit command to end the chat

## 🛠️ Technologies Used

* Python 3
* Conditional Statements (if-else)
* Loops
* Functions
* String Manipulation

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project folder:

   ```bash
   cd Basic-Chatbot
   ```

3. Run the program:

   ```bash
   python chatbot.py
   ```
